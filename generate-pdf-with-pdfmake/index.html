Generate CV with PDFmake

Posted at — Mar 9, 2022

In this personal site I decided to generate my CV myself rather than taking up from URL of another website (e.g you could get generated CV from kalibrr.com).

In my previous project I did the job, generating pdf of documents is pretty simple, there’re some packages you could use out there, one of them is PDFmake.

PDFmake is a package that helps you to generate PDF for server-side and client-side usage in pure JavaScript (read its documentation here), it’s a powerful tool yet easy to use, it has a playground too for you to try it online.

Let’s get some actions.

We’ll create a PDF document of curriculum vitae that includes data of work experiences, skills and interests. To achieve that we have to provide the data and install PDFmake package first.

Create helper function to build document object

 1// to get label of section, there will be 3 section: Experience, Skill and Interest
 2function getSectionLabel(text) {
 3  return { text, style: ["textBlue", "bold"], margin: [0, 0, 0, 5] };
 4}
 5
 6// our sections and their values will be wrapped into tables, so we need the function below to simplify creating the tables
 7function generateTable(body) {
 8  return {
 9    table: {
10      widths: [150, "*"],
11      body,
12    },
13    layout: "noBorders",
14  };
15}

Let’s map our data and generate the table with generateTable() function

 1// generate experiences
 2const generatedExperiences = experiences.map((experience) => {
 3  return [
 4    experience.date,
 5    {
 6      stack: [
 7        { text: experience.position, style: ["bold"] },
 8        { text: experience.company, margin: [0, 0, 0, 5] },
 9        {
10          ul: experience.jobs,
11        },
12        "\n",
13      ],
14    },
15  ];
16});
17const generatedExperienceTable = generateTable(generatedExperiences);
18
19// generate skills
20const generatedSkills = skills
21  .map((skill) => {
22    return ["", { ul: [skill.label] }];
23  });
24const generatedSkillTable = generateTable(generatedSkills);
25
26// generate interests
27const generatedInterests = interests.map((interest) => {
28  return ["", { ul: [interest] }];
29});
30const generatedInterestTable = generateTable(generatedInterests);

The last but not least, create the PDF file and download it. You can just open the file too with .open() function.

1pdfMake.createPdf(dd).download("Awesome CV");
